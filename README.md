# Hospital Management System with AI Assistant
 **Frontend Hospital Management System** developed using **Angular**, integrated with **Groq AI API** for the hospital assistant.<br>
This project helps manage patient records, doctor details, and appointments efficiently using browser localStorage for data storage.

## 🔹 Features

- **Patient Management:** Add, update, view, and delete patient records.
- **Doctor Management:** Maintain doctor details and schedules.
- **Appointments:** Book, track, and manage appointments easily.
- **AI Assistant:** Ask medical-related questions using Groq API.
- **Data Storage:** All data stored in browser localStorage.
- **User-friendly UI:** Responsive and clean interface.

## 🤖 AI Assistant Setup (Groq)

This project uses **Groq AI** for the hospital assistant.<br>
<b>Step 1: Get Groq API Key:</b>
Sign up at https://console.groq.com/ and get your API key.<br>
<b>Step 2: Set API Key:</b> In `src/environments/environment.ts` and `environment.prod.ts`, replace `'your-groq-api-key-here'` with your actual key.
